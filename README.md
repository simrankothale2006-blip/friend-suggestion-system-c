# Social Media Friend Suggestion System using Data Structures in C

## Description
This project implements a **social media friend suggestion system** using
**Data Structures in C**.  
Users are represented as nodes and friendships as edges using **linked lists**.

The system suggests friends based on **mutual connections**, similar to real
social media platforms.

##  Data Structures Used
- Linked List
- Array
- Graph (Adjacency List concept)

##  Functionalities
- Add users
- Create friendships
- Display friend list
- Suggest friends based on mutual friends

##  Input
- User names
- Friendship connections
- Menu-based user interaction

##  Output
- Friend lists
- Suggested friends with mutual friend count

##  How to Run
```bash
gcc friend_suggestion.c -o friend_suggestion
./friend_suggestion
